🚀 Attendance Intelligence — Viyaan AI

A smart, lightweight web application that helps students analyze attendance, predict future eligibility, and make better academic decisions.

---

🎯 Overview

Attendance Intelligence is designed to solve a real student problem:

«“How many classes can I skip?”
“Will I meet attendance criteria?”»

Instead of guessing, this app gives clear, data-driven answers instantly.

---

⚡ Key Features

- 📊 Real-time Attendance Calculation
- 🎯 Minimum Attendance Prediction
- ⏳ Skip / Attend Class Guidance
- 📈 Future Projection Analysis
- 💡 Simple & Fast UI (Mobile Friendly)

---

🧠 How It Works

The system applies basic mathematical logic to compute:

- Current attendance percentage
- Required classes to reach target
- Safe number of classes that can be missed

It transforms raw input into actionable insights, not just numbers.
